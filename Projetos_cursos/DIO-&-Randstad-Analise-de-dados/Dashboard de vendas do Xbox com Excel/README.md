# 🎮 Dashboard de Vendas - Xbox (Excel)

## 📌 Sobre o Projeto

Este projeto consiste na criação de um **Dashboard simples de Vendas de Xbox no Excel**, com foco em visualização clara e objetiva dos principais indicadores comerciais.

O objetivo é transformar dados brutos em informações estratégicas, facilitando a análise de desempenho e a tomada de decisão.

---

## 🛠️ Ferramentas Utilizadas

- Microsoft Excel  
  - Tabelas Dinâmicas  
  - Gráficos Dinâmicos  
  - Segmentação de Dados (Slicers)  
---

Este projeto demonstra como o Excel pode ser utilizado como uma ferramenta eficiente para análise de vendas, mesmo em dashboards simples.

Com organização e visualização adequada, é possível transformar dados em insights estratégicos de forma prática e objetiva.
