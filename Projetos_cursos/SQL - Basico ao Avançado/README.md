# Curso “SQL Básico ao Avançado para Análise de Dados” (Udemy)

Este repositório reúne minhas atividades práticas do curso **"SQL Básico ao Avançado para Análise de Dados"**, ministrado por **André Mendonça Krul** na Udemy

Cada arquivo SQL é um passo na minha jornada de aprendizado — desde consultas simples até técnicas avançadas como CTEs e Window Functions.

##  Estrutura

| Arquivo | Conteúdo |
|---------|----------|
| `01-select.sql` | Comandos básicos: `SELECT`, filtros (`WHERE`), ordenação (`ORDER BY`), funções agregadas |
| `02-joins.sql` | Manipulação de tabelas com diversos tipos de `JOIN` (INNER, LEFT, etc.) |
| `03-cte.sql` | `CTEs` (Common Table Expressions) e consultas mais legíveis |
| `04-window-functions.sql` | Aplicações de Window Functions (`ROW_NUMBER()`, `RANK()`, `OVER()`, etc.) |
| `05-case-examples.sql` | Casos práticos com `CASE WHEN`, transformando e categorizando dados |

##  Objetivos e Aprendizados

- Evoluir na criação de consultas SQL desde o básico até os conceitos mais avançados.
- Aprender a dar estrutura ao código usando **CTEs**, deixando as queries mais limpas e fáceis de entender.
- Aplicar **Window Functions** para análises mais profundas e refinadas.
- Entender e praticar **lógica condicional com `CASE WHEN`**.
- Estar pronto para enfrentar desafios reais de análise de dados do mercado.
